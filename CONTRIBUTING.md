# Commit messages:

This repo uses **`Conventional Commits`** to make commit messages more readable and unified.<br>
***"A specification for adding human and machine readable meaning to commit messages"***

### Quick Summary:

```md
<type>[optional scope]: <description>

[optional body]

[optional footer(s)]
```

### Please read **[Conventional Commits 1.0.0](https://www.conventionalcommits.org/en/v1.0.0/)** before contributing!
